<section id="methodology">

## 工作方法论 / Methodology

在处理实际问题时，我遵循以下闭环路径：

1.  **问题结构化 (Decomposition)** 将模糊场景拆解为清晰的变量、约束条件与目标函数。
2.  **模型设计 (Modeling)** 选择最优数学路径，赋予问题“可计算性”。
3.  **工程实现 (Implementation)** 使用 Python/Excel 快速构建原型并实现自动化验证。
4.  **迭代校准 (Refinement)** 基于偏差分析反馈模型，优化假设，提升预测或分析精度。

</section>
